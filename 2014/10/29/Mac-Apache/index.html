Mac Apache

Mac端 Apache 本地服务器

步骤

第一步：打开终端 输入 sudo apachectl start

第二步：Finder下Cmd+G 输入 /library/WebServer/Documents 添加你所需要的文件 也可用系统自带的文件测试一下 打开后是这样的：

第三步：获取文件读取格式是：http://localhost/＋文件名，如：http://localhost/PoweredByMacOSXLarge.gif
